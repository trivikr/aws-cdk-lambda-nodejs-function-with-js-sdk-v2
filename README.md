# aws-cdk-lambda-nodejs-function-with-js-sdk-v2

Test CDK stack which creates a Lambda Node.js Function which uses JS SDK v2

## Pre-requisites

- Install [Node.js][] 24.x
- Install corepack globally by running `npm i -g corepack`
- Install dependencies by running `pnpm i`

## Setup

- Run `pnpm cdk deploy` deploy this stack to your AWS account/region.

[Node.js]: https://nodejs.org/en/download
